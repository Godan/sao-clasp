> gas [clasp](https://github.com/google/clasp) project template

## Usage

Install [SAO](https://github.com/saojs/sao) first.

```bash
yarn global add sao
# or
npm i -g sao
```

### From npm

```bash
sao clasp my-project
```

### From git

```bash
sao takanakahiko/sao-clasp my-project
```
