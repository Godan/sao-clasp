# template-clasp

> gas clasp project template

## Usage

Install [SAO](https://github.com/egoist/sao) first.

```bash
yarn global add sao
# or
npm i -g sao
```

### From npm

```bash
sao clasp my-project
```

### From git

```bash
sao takanakahiko/template-clasp my-project
```
